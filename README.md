# JCF API Project

A comprehensive RESTful API built with ASP.NET Core following Clean Architecture principles and Domain-Driven Design (DDD).

## Architecture
- **Domain Layer**: Core business entities and logic
- **Application Layer**: Use cases and business rules  
- **Infrastructure Layer**: Data access and external services
- **Presentation Layer**: API controllers and endpoints

## Features
- RESTful API design
- Clean Architecture implementation
- Entity Framework Core ORM
- Dependency injection
- Swagger/OpenAPI documentation
- CORS support
- Error handling middleware

## Technology Stack
- .NET 6.0
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger UI

## API Endpoints
Comprehensive REST API with full CRUD operations for business entities.
